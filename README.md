# Asteroid-Dodger-3D


An immersive, 3D arcade space-flight simulator built using Python and PyOpenGL. Take control of a starship navigating through hazardous asteroid fields, engage hostile celestial elements, and manage vital booster upgrades to survive deep space travel.

## ✨ Features
* **Dynamic 3D Environment:** Renders fluid starry backgrounds, space environments, and detailed 3D spherical planets using OpenGL primitives and lighting maps.
* **Collision & Core Game Loop:** Real-time calculation of physics bounds for shooting down asteroids with space projectiles or dodging obstacles.
* **Ship Enhancements:** Integrated speed-boosting triggers that alter gameplay dynamics, complete with flame visibility adjustments and speed multiplication parameters.
* **Interactive UI States:** Complete interactive menu screens for Start, Pause, and Game Over states alongside life counters and real-time score analytics.
